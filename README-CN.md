## 使用方法
### 每日学习
1. 在本仓库中打开Claude Code
2. 自然提问CFP相关知识点，如同与真人老师对话
3. 回答Claude提出的理解检验问题
4. 每次学习结束后，Claude会自动记录：
   - 本次学习内容
   - 你感到吃力的知识点
   - 已掌握内容
   - 后续需要复习的内容

### 复习回顾
需要复习时，直接向Claude提问：
- “帮我复习一下我掌握不好的知识点”
- “我今天应该重点学什么？”
- “针对我的薄弱环节出题考考我”
- “展示我的学习进度”

Claude会读取你的学习记录，根据过往表现生成个性化复习内容。

### 进度追踪
查看 `/progress/cfp-study-tracker.md` 可全面了解：
- 整体备考完成度（当前82%）
- 已完成的知识模块（4大核心模块 ✅）
- 尚存的知识漏洞
- 考前优先级学习计划

## 学习理念
**引导式学习法：**
- 口语化交流，无评判压力
- 在你现有知识基础上逐步深化
- 确认理解后再推进新内容
- 适配你的个人学习风格
- 注重深度理解，而非单纯死记硬背
